Creating a web portfolio on GitHub is a great way to showcase your projects and skills. You can do this using GitHub Pages, which lets you host a website directly from a GitHub repository. Here's a step-by-step guide:

1. Create a GitHub Account (if you don’t have one)

Go to github.com and sign up.

2. Create a New Repository

  • Click the “+” icon at the top right → New repository.

  • Name the repository:
    Use either yourusername.github.io (for a personal website)
    or any name (for a project portfolio).

  • Choose Public.

  • Check “Add a README file”.

  • Click Create repository.

3. Add Your Portfolio Files

You can:

  • Upload HTML, CSS, JS files manually.

  • Or clone the repo locally and push code using Git.

Example HTML file (index.html):

html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My Portfolio</title>
</head>
<body>
  <h1>Hello, I'm [Your Name]</h1>
  <p>Welcome to my web portfolio.</p>
</body>
</html>

4. Enable GitHub Pages

  • Go to your repository on GitHub.

  • Click Settings > Pages (left sidebar).

  • Under “Source”, choose the branch (usually main) and folder (/root or /docs).

  • Click Save.

  • You’ll see a message with the link to your site (e.g., https://yourusername.github.io/).

5. Customize Your Portfolio

You can enhance your site using:

  • HTML/CSS/JS for custom designs.

  • Templates (e.g., HTML5 UP, Jekyll themes).

6. Keep Updating

Push changes to the repo whenever you want to update your site:

bash
git add .
git commit -m "Update portfolio"
git push

✅ Step 1: Copy the Portfolio Template

📁 Files:

You need two files: index.html and style.css.

  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
